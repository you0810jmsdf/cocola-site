# COCoLa LP

COCoLa 本体サイトのトップページを LP として再構成するための試作プロジェクトです。

GitHub Pages 公開前提で、トップページ、6カテゴリページ、Apps 入口ページ、TOPICS 連携をまとめています。

## ファイル

- `index.html`: LP 本体
- `styles.css`: LP 用スタイル
- `categories/*/index.html`: 6つの「つくる」個別ページたたき台

## 想定用途

- Google Sites の現行トップページ改善前のたたき台
- 埋め込み用の静的プロトタイプ
- 文言整理、セクション構成整理、色味の検討

## GitHub Pages 公開手順

1. GitHub に `cocola-site` リポジトリを新規作成する
2. この `cocola_lp` ディレクトリ配下のファイルをリポジトリ直下へ配置する
3. GitHub の `Settings` から `Pages` を開き、対象ブランチの `/ (root)` を公開元に設定する
4. 公開URLが確定したら、必要に応じて OGP 用の絶対URLや `CNAME` を追加する
5. 公開後にトップ、カテゴリ、Apps、外部リンク、TOPICS 埋め込みを実URLで確認する
